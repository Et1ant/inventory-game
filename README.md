# RPG Inventory System

Консольный менеджер инвентаря на Python, реализующий механику контроля веса предметов. Проект демонстрирует работу со структурами данных и динамическое управление состоянием объекта.

## Функциональность

* **База данных предметов:** Хранение характеристик (название, вес) в формате `dict`.
* **Контроль грузоподъемности:** Динамический расчет текущего веса и блокировка добавления при превышении лимита.
* **Управление содержимым:** Операции добавления (`append`) и удаления (`remove`) элементов списка.
* **Валидация:** Проверка существования предмета в базе данных и наличия в инвентаре.

## Технические особенности

* **Метод `.get()`:** Использование безопасного извлечения данных из словаря для предотвращения ошибок `KeyError`.
* **Обработка строк:** Применение `.lower().strip()` для нормализации пользовательского ввода.
* **Управление памятью:** Использование изменяемого списка (`list`) для хранения текущего набора предметов.
